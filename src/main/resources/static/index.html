This app was built / deployed using Spring Boot, Gradle, Docker, and Amazon Web Service. To check out the source code or build / deployment process, head to https://github.com/clever-wildcard/springboot-gradle-docker-aws.

REST sample: Create, get, replace, delete employees.

Create an employee:

POST

http://annaofalltrades.com/employees

{'name': 'Anna Soto', 'role': 'Software Engineer'}

Get all employees:

GET

http://annaofalltrades.com/employees

Get a single employee:

GET

http://annaofalltrades.com/employees/{id number}

Replace an employee:

PUT

http://annaofalltrades.com/employees/replace/{id number}

{"name": "Curious the Cat", "role": "Companion"}

Delete an employee:

DELETE

http://annaofalltrades.com/employees/{id number}
